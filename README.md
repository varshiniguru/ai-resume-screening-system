# AI Resume Screening System

## Project Overview

AI Resume Screening System is a Flask-based web application that helps recruiters upload resumes, analyze candidate skills, calculate job match scores, rank candidates, and export reports.

---

## Features

* User Authentication
* Resume Upload (PDF)
* Skill Extraction
* AI Resume Matching
* Candidate Ranking
* Dashboard UI
* PDF Report Export
* Excel Report Export

---

## Technologies Used

* Python
* Flask
* HTML
* Bootstrap
* Pandas
* ReportLab

---

## Installation Guide

### Step 1

Clone the repository:

git clone https://github.com/varshiniguru/ai-resume-screening-system.git

### Step 2

Open project folder.

### Step 3

Create virtual environment:

python -m venv venv

### Step 4

Activate virtual environment:

venv\Scripts\activate

### Step 5

Install dependencies:

pip install flask pandas reportlab openpyxl

### Step 6

Run the project:

python app.py

### Step 7

Open browser:

http://127.0.0.1:5000

---

## User Manual

### Login

Users can log in using username and password.

### Resume Upload

Upload PDF resumes through the upload form.

### AI Matching

The system analyzes resume skills and compares them with job descriptions.

### Ranking

Candidates are ranked based on their match score.

### Reports

Users can export reports in PDF and Excel format.
